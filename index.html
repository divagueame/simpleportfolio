TurboNews

TurboNews is an extended and advanced version of one of my first Rails apps: a basic web scraper which crawls the newspapers. It automatically gathers the headlines every 6 hours and lists the links to the user with the articles of the day. TurboNews works as an automated tool to search, index and organize different articles and topics. I keep developing and improving it on a regular basis.

    Built with:
  • Ruby on Rails
  • PostgreSQL
  • Tailwind CSS
    Relevant features:
  • Live Search updates for articles
  • Administration Panel
  • Pagination
  • Wide coverage testing with MiniTest/Capybara
  • Automatized assignation of category tags to each article based on word frecuency

New Summit

NewSummit is an application to keep track of running time with the scope of achieving new goals and with the purpose of building the consistency of daily practice. Users are given a goal by default upon registration, but they can set their own personal goal and the deadline to accomplish it. The program will then suggest the appropriate daily goal to achieve the goal and help logging the running sessions. I plan to add the support for several users to share their goals and see each others' progress.

    Built with:
  • Ruby on Rails
  • PostgreSQL
    Relevant features:
  • User authentication and authorization with Devise

ChikiDeploy

ChikiDeploy is a CLI script written in Bash to extend the Rails app generator. It allows to use a single command to create a new Rails application on a virtual private server running on Linux Ubuntu. It leverages the power of GitHub CLI and Mina to deploy the code on the server and get it ready for a quick deployment.

    Built with:
  • Bash
  • GitHub CLI
  • SSH
  • Mina/Capistrano
