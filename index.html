TurboNews

TurboNews is an extended and advanced version of one of my first Rails apps: a basic web scraper which crawls the newspapers. It automatically gathers the headlines every 6 hours and lists the links to the user with the articles of the day. TurboNews works as an automated tool to search, index and organize different articles and topics. I keep developing and improving it on a regular basis.

    Built with:
  • Ruby on Rails
  • PostgreSQL
  • Tailwind CSS
    Relevant features:
  • Live Search updates for articles
  • Administration Panel
  • Automatized assignation of category tags to each article based on word frecuency
